# ghost-theme-artshow

**小梦岛 · Underground Gallery** — A Ghost theme where every post is a unique artwork.

每篇文章都是一件展品，博客即艺术展览。

---

## 概念 / Concept

传统博客所有文章共用同一套 UI。这个主题打破了这个惯例——每篇文章的页面风格由 AI 独立生成，布局、配色、字体排版与文章内容深度结合。读者每次点进一篇文章，都会看到一个全新的视觉世界。

Traditional blogs use the same UI for every post. This theme breaks that convention — each post's page style is independently generated by AI, with layout, colors, and typography deeply tied to the content. Every article is a unique visual experience.

---

## 特性 / Features

- 地下画廊风格首页，暗色系艺术展览大厅
- 每篇文章可拥有完全独立的 AI 生成页面
- 支持普通文章模式（默认主题布局）与 AI 全量接管模式
- AI 生成前自动分析文章风格并给出推荐，用户可直接采用或修改
- 分类页、独立页面完整支持
- 附带 AI 生成脚本，一条命令生成并写回 Ghost

---

## 环境要求 / Requirements

- Ghost >= 6.0.0
- Node.js >= 18（用于 AI 生成脚本 / for AI generation script）
- OpenAI API Key（或兼容 OpenAI 接口的服务）

---

## 安装主题 / Install Theme

1. 下载 `xiaomengtao-theme.zip`
2. Ghost Admin → Settings → Design → Change theme → Upload theme
3. Activate

---

## 配置 AI 生成脚本 / Setup AI Script

```bash
cd scripts
npm install
cp .env.example .env
```

编辑 `.env` / Edit `.env`:

```
GHOST_URL=https://your-ghost-site.com
GHOST_ADMIN_API_KEY=your-admin-api-key
OPENAI_BASE_URL=https://api.openai.com/v1
OPENAI_API_KEY=your-api-key
OPENAI_MODEL=gpt-4o
```

> Ghost Admin API Key: Ghost Admin → Settings → Integrations → Add custom integration

> `OPENAI_BASE_URL` 支持任何兼容 OpenAI 接口的服务（如 Azure、本地模型等）

---

## 使用 / Usage

写完一篇文章后，运行脚本生成 AI 页面：

```bash
# 完整流程：AI 分析风格 → 生成 HTML → 本地预览 → 确认写回
node scripts/generate-ui.js your-post-slug

# 直接上传本地写好的 HTML
node scripts/generate-ui.js your-post-slug --file path/to/page.html

# 只生成 prompt 文件，不调用 AI
node scripts/generate-ui.js your-post-slug --prompt-only
```

默认模式脚本会自动：

1. 拉取文章内容
2. **AI 分析文章，推荐视觉风格**（可回车采用或手动修改）
3. 调用 AI 生成完整 HTML 页面
4. 保存本地预览文件，确认后写回 Ghost
